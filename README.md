# Malaria Parasite Prediction Using CNN

## Project Overview

This project focuses on the automated detection of malaria parasites from microscopic cell images using a Convolutional Neural Network (CNN). The model classifies cell images into two categories: Parasitized and Uninfected.

## Objective

The objective of this project is to develop a deep learning model that assists in the early detection of malaria by analyzing microscopic blood cell images. Automated detection can help reduce manual effort and improve diagnostic efficiency.

## Dataset

The dataset consists of microscopic images of red blood cells categorized into:

* Parasitized Cells
* Uninfected Cells

Total Images: Approximately 27,500+

The CNN model successfully classifies malaria-infected and uninfected cell images with high accuracy and demonstrates the effectiveness of deep learning in medical image analysis.
